📝 Passo a Passo Simplificado

Passo 1: Criar Planilha Google Sheets

Estrutura básica com colunas: Produto, Link, Descrição, Imagem, Preço, Status, Hora

⏱️ Tempo: 10 minutos
🎯 Objetivo: Base de dados das ofertas

Passo 2: Configurar Evolution API no Railway

Deploy automático, conectar WhatsApp via QR Code, pegar credenciais

⏱️ Tempo: 15 minutos
🎯 Objetivo: Conexão com WhatsApp

Passo 3: Pegar IDs dos Grupos WhatsApp

Usar o Manager da Evolution API para listar grupos e copiar IDs

⏱️ Tempo: 5 minutos
🎯 Objetivo: Saber onde enviar

Passo 4: Hospedar Imagens

Upload no Imgur ou Google Drive, copiar links públicos

⏱️ Tempo: 10 minutos
🎯 Objetivo: Imagens visíveis nas mensagens

Passo 5: Criar Cenário no Make.com

Conectar Google Sheets → Iterator → HTTP (WhatsApp) → Update Sheets

⏱️ Tempo: 30 minutos
🎯 Objetivo: Automação completa

Passo 6: Testar e Ativar

Executar teste manual, verificar funcionamento, ativar schedule

⏱️ Tempo: 10 minutos
🎯 Objetivo: Sistema no ar! 🎉

⚠️ Dicas Para Evitar Problemas

🚫 NÃO faça:
• Enviar mais de 10 msgs/dia no mesmo grupo
• Usar sempre a mesma mensagem (varie o template)
• Enviar em horários muito cedo (antes 8h) ou tarde (depois 22h)
• Esquecer de pausar em feriados/fins de semana
✅ FAÇA:
• Varie os horários (não envie sempre no mesmo)
• Adicione delay de 3-5 segundos entre envios
• Teste primeiro em um grupo de teste
• Monitore a planilha semanalmente
• Use mensagens autênticas e úteis
